= SHIPPU GSS NN =
SHIPPU GSSはタイムアタックシューティングをベースにしたニューラルネットワークに
よる自動プレイの実験です。

= 実行方法 =
Python、Pygameをインストール後、gss.pyを実行してください。
実行中は世代交代時に遺伝子情報が記録されたpickleファイルを出力します。起動時に
引数でこのファイルを指定することにより途中から継続して実行することができます。

= 操作方法 =
 * Escキー ... ゲーム終了 / タイトル画面に戻る
